>  Command line client


## Install

```sh
$ npm install --save kms-cli
```


## Usage

```js
var kmsCli = require('kms-cli');

kmsCli('Rainbow');
```

```sh
$ npm install --global kms-cli
$ kms-cli --help
```
```docker
alias kms-cli=./run-cli.sh
```
